## GitHub为何物

# 关于GitBook

## GitBook为何物
`GitBook` 是一款工具，它可以让你把用 `markdown` 写的文档转换成 `html` 或 `pdf` 格式，方便阅读  
`GitBook` 主要由两个组件组成
* 一个名为 `GitBook` 的命令行工具
* `gitbook.com`,一个 `GitBook` 专用的托管服务
